## Development

- Prerequisite

  + [x] [Latest Node LTS](https://nodejs.org/en/download/) or via [NVM](https://github.com/nvm-sh/nvm/blob/master/README.md)
  + [x] [Yarn](https://yarnpkg.com/lang/en/docs/install/)

- Install deps

```bash
yarn
# or
npm i
```

- Running in local

```bash
yarn dev
# or
npm run dev
```

- Build for production

```bash
yarn build
# or
npm run build
```
